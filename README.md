# Landing Page Imobiliária

## 🚀 Projeto criado com:

- **React 18**
- **Vite**
- **Tailwind CSS**

## 📦 Instalação

```bash
npm install
```

## 🏃‍♂️ Executar o projeto

```bash
npm run dev
```

## 🎨 Cores configuradas:

- **Azul**: #1e40af (primary-blue)
- **Dourado**: #f59e0b (primary-gold)
- **Cinza**: #64748b (primary-gray)

## 📁 Estrutura:

```
src/
├── App.jsx      # Componente principal
├── main.jsx     # Entry point
└── index.css    # Estilos + Tailwind
```
